# Xingyu-Bot

一個集結許多功能的DiscordBot

- Discord群組管理、整合各群組的跨群警告紀錄、成員加入、離開通知、語音頻道紀錄
- 各類系統：經濟系統、寵物系統、音樂系統、類似Twitch實況預測的賭盤系統
- 投票、抽獎、讓機器人選擇等群組輔助功能
- 各項自動通知，包含地震報告、天氣預報
- Youtube開台、新影片通知；Twitch開台、新影片、新剪輯通知；Twitter/X推文通知
- 支援Osu、Riot、Steam、Twitch、Apex等API
- 與Twitch chat bot連動資料
- 整合的遊戲玩家資訊資料庫
- 使用PostgreSQL作為資料庫
- 私人功能：Minecraft伺服器管理
